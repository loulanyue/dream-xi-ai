| 包 | 状态 | 说明 |

| `@dream-xi/router` | ✅ 完成 | A2A 路由（@mention + 意图推断 + 线程管理） |
| `@dream-xi/server` | ✅ 完成 | HTTP 服务层（11 个端点，完整中间件链） |

**API 端点清单：**

```
GET  /health                    — 服务健康 + 球员上场状态
GET  /api/health                — 同上（别名）
GET  /api/players               — 所有球员状态
GET  /api/players/:id           — 单名球员状态
POST /api/chat                  — 聊天（核心端点，10 步流水线）
GET  /api/threads               — 线程列表
POST /api/threads               — 创建新线程
POST /api/threads/:id/archive   — 归档线程
GET  /api/memory/:playerId      — 查询球员情景记忆
GET  /api/fair-play/stats       — 铁律守卫统计
```
